# easymd 主题 CSS 变量清单

以下是 easymd 主题系统使用的所有 CSS 变量。你可以在「手写 CSS 模式」下修改这些变量来自定义主题。

## 全局

| 变量名                     | 说明                | 示例值                     |
| -------------------------- | ------------------- | -------------------------- |
| `--easymd-page-padding`      | 页面左右内边距      | `20px`                     |
| `--easymd-font-size`         | 正文字体大小        | `16px`                     |
| `--easymd-line-height`       | 正文行高            | `1.7`                      |
| `--easymd-paragraph-margin`  | 段落上下间距        | `16px`                     |
| `--easymd-paragraph-padding` | 段落上下内边距      | `0px`                      |
| `--easymd-text-color`        | 正文颜色            | `#333`                     |
| `--easymd-primary-color`     | 主题色              | `#1e6bb8`                  |
| `--easymd-primary-color-20`  | 主题色 (透明度 12%) | `rgba(30, 107, 184, 0.12)` |
| `--easymd-primary-color-30`  | 主题色 (透明度 18%) | `rgba(30, 107, 184, 0.18)` |
| `--easymd-primary-color-50`  | 主题色 (透明度 50%) | `#1e6bb850`                |
| `--easymd-letter-spacing`    | 全局字间距          | `0px`                      |
| `--easymd-underline-style`   | 下划线样式          | `solid`                    |
| `--easymd-underline-color`   | 下划线颜色          | `currentColor`             |
